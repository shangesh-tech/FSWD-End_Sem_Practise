```
Contact Book App
Problem Statement
Create a simple React app to manage contacts.
Requirements:
● Add Contact: Enter name, phone number, and email.
● Edit Contact: Update contact information.
● Delete Contact: Remove a contact.
● Search: Search contacts by name.
● Display: Show list of all contacts.

Technical Expectations:
● Components: ContactForm, ContactList, SearchBar.
● Manage all state with useState.
● No backend, frontend only
```
